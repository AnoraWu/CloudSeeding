### Power
This folder was used for statistical power analysis (Table S1 in the paper draft).

### 抗议数据 (Protest Data)
This folder cleaned Weibo posts, constructed panel data, and analyzed whether the Chinese government uses cloud seeding as a tool to control protests. The results were not included in the paper but were presented in presentations.

### 气象局数据 (Meteorological Bureau Data)
This folder processed official weather bureau records. These cloud seeding records were added to the Weibo cloud seeding data for all of the regression analyses.

### 灾害数据 (Disaster Data)
This folder cleaned natural disaster records and analyzed whether cloud seeding helped reduce hail frequency (Table S10 in the paper draft).
