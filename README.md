### Power: 
This folder was used for statistical power analysis (Table S1 in the paper draft).

### 抗议数据  (Protest Data)
This folder cleaned weibo posts, constructed panel data, and analyzed the if Chinese goverment is using cloud seeding as a tool to control protests. The results were not included in the paper but presentations. 

### 气象局数据 (Meteorological Bureau Data)
This folder processed official weather bureau records. Those cloud seeding records were added to the weibo cloud seeding data for all the regression analysis. 

### 灾害数据/ (Disaster Data)
This folder cleaned natural disaster records and analyzed if cloud seeding helped reduce hail frequencies (Table S10 in the paper draft).
